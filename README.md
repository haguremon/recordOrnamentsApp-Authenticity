## 機能一覧
* 新規登録
* ログイン機能
* 置き場所のCRUD
### 細かな機能
* パスワードの有無によってエフェクトをつける

## 環境
* Xcode 13.1
* Swift 5.5.1
## 導入したライブラリ
※cocoapodを利用してライブラリの管理を行なってましたが、使いづらいのでSwift Package Managerに移行して管理をしてます。

* Firebase
* Firebase/Firestore
* SDWebimage
* SideMenu
* JGProgressHUD
* lottie-ios
